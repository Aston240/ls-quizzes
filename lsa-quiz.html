One-Time Setup

1. Go to github.com/Aston240/ls-quizzes → Settings → Pages → Source: main branch, root. Save.
2. Create a folder called quizzes in the repo.
3. Upload this file (lsa-quiz.html) to the root of the repo.
4. For each new quiz, upload its JSON file to the quizzes/ folder, then use the builder above.

Your quiz link will look like:
https://aston240.github.io/ls-quizzes/lsa-quiz.html?q=...
